Sends email notifications to users upon login.

== Description ==
This plugin will send Email notification to the user upon login to site. Once your logged in to the WordPress site than it will send an email to user's registered email in site along with date, time, IP address and user agent.

**Features**:
* Send Email notification upon login into WordPress
* Free and Open Source 
* Lightweight and Secure
* No need to any configurations, directly work on activations.


== Installation ==

1. Go to Plugins > Add New Plugin section on your WordPress site
2. Click on "Upload Plugin", choose the plugin zip file and tap the "Install Now" button.
2. Activate the "Login Alert for WordPress" plugin.
